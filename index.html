ศึกษาหลักสูตรพัฒนาวิชาการ ทักษะคอมพิวเตอร์ และเทคโนโลยีการศึกษา พร้อมทั้งมีส่วนร่วมในกิจกรรมฝึกฝนวิชาชีพที่เน้นการนำซอฟต์แวร์และการเขียนโปรแกรมประยุกต์มาพัฒนาท้องถิ่น

ภูมิหลังการศึกษา

การศึกษาขั้นพื้นฐาน

สถาบันการศึกษาในเขตพื้นที่จังหวัดพังงา

สร้างพื้นฐานความรู้ทางด้านวิทยาศาสตร์ คณิตศาสตร์ เทคโนโลยีและคอมพิวเตอร์พื้นฐาน รวมถึงการร่วมทำกิจกรรมขับเคลื่อนกลุ่มเยาวชนพัฒนาชุมชนบางม่วง อำเภอตะกั่วป่า

ทักษะและความสามารถ

การใช้งานเชิงเทคนิค ทักษะร่วมสมัย และสมรรถนะการจัดแสดงงานในที่สาธารณะ

การพัฒนาเว็บแบบ Responsive

ความเชี่ยวชาญการใช้ HTML, CSS (Tailwind) และโมเดล JavaScript ในการเขียนโค้ดและขึ้นรูปแบบโปรไฟล์ที่ทำงานได้อย่างลื่นไหลในมือถือ แท็บเล็ต และเดสก์ท็อป

ระดับความเชี่ยวชาญ ดีเยี่ยม (Professional)

การนำเสนอและสื่อสารสารสนเทศ

การจัดวางโครงสร้างสไลด์ เทคนิคการจัดเรียงพอร์ตโฟลิโอ และการอ้างอิงข้อมูลอย่างเป็นระเบียบที่เหมาะกับการเสนองานในสถานที่ประชุมภายนอก

ระดับความเชี่ยวชาญ ระดับสูง (Advanced)

โครงสร้างเว็บประยุกต์แบบหลายธีม

การสร้างตัวเลือกตัวแปรสีสไตล์ CSS Variables ที่ให้สลับเปลี่ยนอารมณ์ของสีได้อย่างทันทีถึง 10 รูปแบบสี พร้อมการทำงานสลับดาร์กโหมด
